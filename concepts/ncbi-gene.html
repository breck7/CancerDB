NCBI Gene

id ncbi-gene name NCBI Gene type onlineDatabase website https://www.ncbi.nlm.nih.gov/gene parentOrganization ncbi description The Gene database provides detailed information for known and predicted genes defined by nucleotide sequence or map position. Currently, Gene contains over 33 million entries and includes data from all major taxonomic groups. Each record in the database corresponds to a single gene and is derived from processing by the NCBI Reference Sequence and genome annotation groups. You can access data from the Gene database on the web through the Gene homepage, programmatically through the Entrez Programming Utilities, or by file transfer through its FTP site.
